# Suggested Commands

## Build & Run
```bash
npx tsc --noEmit          # Type check
npx tsc                    # Compile
npx tsx src/cli.ts         # Run CLI directly (dev mode)
```

## Testing
```bash
npx tsx test_scripts/<test-file>.ts   # Run individual test
```

## Package Management
```bash
npm install                # Install dependencies
npm init -y                # Initialize project (first time)
```

## System Utils (macOS/Darwin)
```bash
git status                 # Check git status
ls -la                     # List files
grep -r "pattern" .        # Search in files
find . -name "*.ts"        # Find TypeScript files
```
