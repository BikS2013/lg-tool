# Task Completion Checklist

When a task is completed, run the following:

1. **Type check**: `npx tsc --noEmit`
2. **Run tests**: Execute relevant test scripts in `test_scripts/`
3. **Update documentation**:
   - Update `CLAUDE.md` if new tools were created
   - Update `docs/design/project-design.md` if design changed
   - Update `docs/design/project-functions.md` if new features added
   - Update `Issues - Pending Items.md` - remove fixed issues, add new ones
4. **No version control** operations unless explicitly requested
